# `packages/types`

Shared contracts for the OneRhythm MVP domain.

This package contains:

- documented TypeScript types for shared domain models
- JSON Schema files for validation and cross-language use

Educational content schemas intentionally exclude ECG-derived clinical inputs.
They also reserve explicit space for disclaimer enforcement and source traceability.

Research Pulse contracts define the reviewed article feed and its provenance-first
translation format for public and profile-filtered educational retrieval.
